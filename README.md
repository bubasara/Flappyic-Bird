# Flappyić Bird
## Flappyić Bird is a Python desktop game similar to the world-famous game Flappy Bird, made as a college project in collaboration with my colleagues.
### "ić" in flappy refers to a little flappy bird (in Serbian)

#### What's used?
* Pygame
